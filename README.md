# 🚀 TodoList em React (Projeto de Estudo)

![Preview do Projeto]('./')

Este projeto é uma **aplicação simples de gerenciamento de tarefas (Todo List)** desenvolvida utilizando **React**.

Ele foi criado como **meu primeiro projeto de prática com React**, com o objetivo de aprender na prática como funciona a construção de aplicações modernas utilizando componentes, estados e comunicação com APIs.

Embora seja um projeto simples, ele representa **um passo importante na construção da minha base como desenvolvedor Front-End.**

---

# ✨ Funcionalidades

✔️ Adicionar novas tarefas
✔️ Marcar tarefas como concluídas
✔️ Alternar status da tarefa (pendente/concluída)
✔️ Remover tarefas da lista
✔️ Interface simples, clara e organizada

O objetivo foi simular o comportamento de uma aplicação real de produtividade.

---

# 🧠 O que eu pratiquei neste projeto

Durante o desenvolvimento deste projeto, pratiquei conceitos importantes do ecossistema React:

* Estruturação de aplicações com **componentes reutilizáveis**
* Gerenciamento de **estado com React Hooks**
* Simulação de **API backend utilizando MirageJS**
* Consumo de dados com **requisições HTTP usando Axios**
* Estilização utilizando **styled-components**
* Organização de projeto utilizando **Vite**

Essas tecnologias fazem parte do fluxo moderno de desenvolvimento de aplicações web.

# 💻 Rodando o projeto localmente

Clone o repositório e instale as dependências:

```bash
npm install
```

Depois inicie o projeto:

```bash
npm run dev
```

O projeto será iniciado normalmente em:

```
http://localhost:5173
```
